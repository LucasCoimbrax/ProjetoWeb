# ProjetoWeb

### 1º: Instalando o Repositório
    
    1. Clone o repositório do Github;

    2. Instale as dependências máquina:

        `npm i`

    3. Rode o programa utilizando:

        `npm run start`

### 2º: Criando o Banco de Dados no Microsoft SQL Server
